## 使用说明

---

### 一、三种工作模式

按一下编码器按键，模式按 **OFF → MANUAL → AUTO → OFF** 循环切换。

| 模式 | 状态 | 行为 |
|:--:|:--:|------|
| **0 — OFF** | 🔴 | 强制关灯，拧旋钮无反应 |
| **1 — MANUAL** | 🟡 | 旋钮直接控制亮度，传感器不参与 |
| **2 — AUTO** | 🟢 | 全自动：有人开灯、无人关灯、自动调亮度 |

---

### 二、模式 0 — OFF（关灯）

```
上电默认进入此模式。
灯保持熄灭状态，无论是否有人、是否有声音。
```

- PWM 输出 = 0，LED 完全熄灭
- 旋转编码器无效
- PIR、声音传感器被忽略

---

### 三、模式 1 — MANUAL（手动调光）

```
按 1 次按键从 OFF 进入。
灯光亮度完全由你手动控制。
```

- **顺时针拧** → 亮度增加
- **逆时针拧** → 亮度降低
- 拧满一圈 = 从最暗到最亮（约 20 级步进）
- 亮度上限受硬件限制（PWM 最大值的 20%，防过流）
- 无超时限制，永久保持你设定的亮度
- PIR、声音、LDR 传感器均不参与

---

### 四、模式 2 — AUTO（全自动）

```
按 2 次按键从 OFF 进入。
完全自动化，无需任何操作。
```

#### 4.1 开灯条件

> **只有 PIR 检测到人体时才会开灯。**

| PIR（有人） | 声音 | 动作 |
|:--:|:--:|------|
| ✅ 检测到 | 任意 | 🟢 **开灯** + 自动调亮度 |
| ❌ 未检测到 | ✅ 有声 | 🔴 保持灭（声音不能开灯） |
| ❌ 未检测到 | ❌ 无声 | 🔴 保持灭 |

> 设计意图：门外噪音、隔壁声音不会误开灯。

#### 4.2 维持亮灯条件

> **灯亮后，PIR 或声音任一检测到活动，都会刷新"无人计时器"。**

| 场景 | 效果 |
|------|------|
| 人在房间走动 | PIR 持续检测 → 计时器不断刷新 → 灯常亮 |
| 人坐着不动 | PIR 无输出，但说话/打字/咳嗽 → 声音续命 |
| 人短暂离开（<5 秒） | 回到房间时计时器未到期 → 灯仍亮 |

#### 4.3 关灯条件

> **连续 5 秒既无 PIR 信号也无声音信号 → 自动关灯。**

```
灯亮中：
  每一轮（50ms）检查 PIR 和声音
    ├─ 任一检测到活动 → 重置 5 秒倒计时
    └─ 两者均为 0 → 倒计时继续
         └─ 满 5 秒 → 🔴 关灯
```

#### 4.4 自动亮度调节

灯亮期间，系统持续读取 LDR 光敏电阻：

- **环境越暗** → 灯光越亮
- **环境越亮** → 灯光自动调暗
- 亮度变化有滞回（防 ADC 噪声导致闪烁）

#### 4.5 自动切手动

**在自动模式下拧动旋钮 → 立即切换到模式 1（手动），亮度 = 当前旋钮位置。** 这是一种快速接管方式。

---

### 五、关键参数

| 参数 | 值 | 说明 |
|------|:--:|------|
| 自动关灯超时 | 5 秒 | 无人+无声后等待多久关灯 |
| 主循环周期 | 50ms | 传感器采样和逻辑更新频率 |
| 声音去抖 | 100ms | 需连续 2 次确认才判定有声 |
| LDR 调光滞回 | 50（ADC 值） | 环境光变化小于此值不调亮度 |
| 亮度上限 | PWM 的 20% | 硬件保护，防止 LED 过流 |

---

### 六、典型场景走查

| 场景 | 系统行为 |
|------|------|
| 上电 | 模式 0（OFF），灯灭 |
| 按 1 次按键 | 模式 1（MANUAL），拧旋钮调亮度 |
| 按 2 次按键 | 模式 2（AUTO），PIR 检测到人 → 灯自动亮 |
| 人离开房间 | 5 秒后灯自动灭 |
| 人在房间但静止不动 | 说话/打字声触发声音传感器 → 维持亮灯 |
| 门外有人经过 | PIR 未检测到（红外不穿透墙壁）→ 灯不亮 |
| 门外有噪音 | 声音不能开灯（仅 PIR 可开灯）→ 灯不亮 |
| 自动模式下拧旋钮 | 立即切到手动模式，亮度由旋钮决定 |
| 再按按键 | 回到 OFF 模式 |
